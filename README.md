# 🏠 家庭用品管理小程序

轻量、温暖的家庭资产管理工具。记录家中每一件物品的名称、规格、购买信息，帮你了解家底、避免重复购置。

---

## 功能特性

| 模块 | 功能 |
|------|------|
| **首页仪表盘** | 物品总数 / 总价值 / 本月新增，分类一览 |
| **分类浏览** | 8大分类，标签/价格/时间排序，瀑布卡片展示 |
| **搜索** | 全局搜索（名称/品牌/标签/渠道），搜索历史 |
| **物品详情** | 规格参数、购买信息、持有时长、编辑/删除 |
| **添加物品** | 3步骤引导：基本信息 → 规格参数 → 购买信息 |
| **设置页** | 分类统计图、数据导出/导入、清空数据 |

---

## 数据结构

每件物品记录以下字段：

**基本信息**
- 名称（必填）
- 分类（必填）
- 存放位置
- 标签（最多5个）

**规格参数**
- 品牌 / 型号 / 颜色 / 尺寸容量 / 重量 / 材质面料

**购买信息**
- 购买日期（必填）
- 购买价格（必填）
- 购买渠道

**附加**
- 备注说明
- 照片（可扩展）

## 快速开始

1. 打开微信开发者工具
2. 导入项目，选择 `projects/family-inventory` 目录
3. 填入 AppID（可使用测试号 `touristappid`）
4. 点击「编译」即可运行

## 设计风格

- **色调**：森绿 #3D7A5C + 暖米白 #FAFAF7 + 金棕点缀 #C88A2C
- **字体**：系统字体栈（PingFang SC / Microsoft YaHei）
- **风格**：温暖极简，像一本整理得很好的生活手册
- **交互**：浮动添加按钮、步骤引导表单、滑动分类标签栏

---

## 技术栈

- 微信小程序原生框架（MINA）
- WXML + WXSS + JavaScript
- 本地存储（wx.setStorageSync）
- 无需后端，完全离线运行
